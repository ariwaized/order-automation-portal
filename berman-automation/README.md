# 🍞 אוטומציה להזמנות מברמן

## התקנה (פעם אחת בלבד)

```bash
cd berman-automation
npm install
npx playwright install chromium
```

## הפעלה

```bash
node berman-order.js order.json
```

## מבנה קובץ ה-JSON

```json
{
  "date": "28-06-2026",   ← ריק = מחר אוטומטית
  "items": [
    { "sku": "101",  "name": "לחם אחיד פרוס", "quantity": 50 },
    { "sku": "238",  "name": "ברמן אסלי פיתות", "quantity": 20 }
  ]
}
```

## זרימת האוטומציה

```
1. 🔐 כניסה (23987 / bikurim2023)
2. ❌ סגירת הודעת הפופ-אפ
3. 📅 לחיצה על שורת התאריך → לחיצה על "צפייה"
4. 🛒 כניסה לקטלוג (הוסף פריט) → חיפוש כל מק"ט → הגדרת כמות
5. 🏠 חזרה לדף הבית → בחירת תאריך → "צפייה"
6. ⏸️  עצירה לאישורך לפני "שדר הזמנה"
7. ✅  שיגור לאחר לחיצת Enter
```

## הערות בטיחות
- הסקריפט **עוצר לפניך פעמיים** לפני ביצוע
- הדפדפן **נפתח ונראה לעין** – אתה רואה הכל
- **Ctrl+C** בכל שלב = ביטול מיידי
